Day9

第一题

题目内容：寻找两个正序数组的中位数

这题如果不考虑复杂度，归并排序即可，但是复杂度为O(m+n)，我们希望得到O(log（m+n)）的算法。因此我们考虑使用二分查找。

将两个数组分为左右两个部分。

1、当两个数组总长度为偶数

左右长度相同，左部分最大值小于右部分最小值，中位数为两者平均。

2、总长度为奇数

左比右大1，中位数是左边最小值

A:0 1 … i-1 | i … m - 1

B:0 1… j-1|j … n - 1

B [ j - 1 ] > A [ i ]
我们需要增加 i ，为了数量的平衡还要减少 j ， j = ( m + n + 1) / 2 - i，i 增大，j 自然会减少。

A [ i - 1 ] > B [ j ]
我们要减少 i ，增大 j 。
上边两种情况，我们把边界都排除了，需要单独讨论。

当 i = 0, 或者 j = 0，也就是切在了最前边。
当 j = 0 时，最大的值就是 A [ i - 1 ] ；当 i = 0 时 最大的值就是 B [ j - 1] 。右半部分最小值和之前一样。

当 i = m 或者 j = n，也就是切在了最后边。
此时左半部分最大值和之前一样。右半部分当 j = n 时，最小值就是 A [ i ] ；当 i = m 时，最小值就是B [ j ]
